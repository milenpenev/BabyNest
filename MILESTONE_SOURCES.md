# Milestone content sources

Verified **2026-07-22**. Content is stored locally and versioned as `CDC-LTSAE-2026.1`; no source is fetched at runtime.

- [CDC — Learn the Signs. Act Early. milestones](https://www.cdc.gov/act-early/milestones/index.html), updated 2026-02-16. Main family-facing catalog source. CDC describes these as skills most children do by an age; BabyNest does not turn missing records into failure or diagnosis.
- [WHO — Child Growth Standards, motor development milestones](https://www.who.int/tools/child-growth-standards/standards/motor-development-milestones). Context for broad motor achievement windows only; BabyNest does not calculate a motor score.
- [NHS — Early learning and development](https://www.nhs.uk/best-start-in-life/early-learning-development/). Neutral activity and communication guidance.

Limitations: parent observations are not a validated developmental screen or clinical assessment. Individual sequences vary; alternative mobility is valid and crawling is not treated as universally required. Corrected age is displayed only as informational context through 24 months.
